产品介绍
PRODUCT DESCRIPTION
产品功能
Product Features
运营监管
资产、合同、司机各类业务的情况一览无遗
业务统计： 有多少司机、在库车辆多少、运营合同多少?实时更新
缴租&合同：新签合同、该收租金、多少滞纳、合同结算，合同的履约全电子记录
工作提醒：违章、保险、事故等等，今天要干的事一个也没少
钱款支付
支持微信公众号在线支付渠道，并自动关联当月租金和管理费
租金缴纳：根据与司机签到的合同情况，提前提醒到每一位司机和公司管理人员
合同收租： 租金、定金、首付款、等等各类钱款的支付金额、时间均有据可查
设备管理 (GPS、OBD)
每台车安装的是何种设备，费用是否过期，车辆是否失踪？
装车设备：管理各类设备的装车情况，跟踪设备SIM卡到期时间
车辆监控：车辆越界与否，去向何方，行程回放，失踪提醒
违章检测
中国最准确的违章信息发布渠道
违章管理：系统在每月的固定时间，自动完成车辆违章数据的查询，并提醒到司机和公司管理者
短信推送
老板不在公司也可以知道公司的运营情况，司机再远也能收到短信提醒
短信报表： 老板的智能秘书，每日公司的运营统计短信直到手机
驾照提醒： 驾驶证、上岗证、年检提醒、各种提醒自动推送到司机的手机
智能移动端
外勤没有网？电脑背着太重？照片记录没地存。
司机/车辆证照： 帮助管理者通过手机一键存储数据到平台
工作提醒： 外勤保险、事故，第一时间将实时情况汇报到公司
